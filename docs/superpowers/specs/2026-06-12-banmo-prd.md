## 3. 도메인 모듈 현황

### 3.1 사용자 (Users)

**구현 완료**
- 회원가입: 카카오 OAuth + 이메일
- 프로필: 닉네임, 소개, 경력, 악기, 지역, 유튜브 링크, 첨부파일(이력서)
- 공개 설정: 소개/경력/첨부파일/악기/지역 항목별 공개 여부 토글
- 전화번호 수집 컬럼 (저장만, UI 미완)
- 탈퇴(soft delete: `deletedAt`)
- 차단 (`block.entity.ts`)

**음표 신뢰도 등급 시스템**

| 등급 | 조건 | 아이콘 |
|---|---|---|
| SIXTEENTH (16분음표) | 신규 / 0점 | - |
| EIGHTH (8분음표) | 10점 이상 | - |
| QUARTER (4분음표) | 30점 이상 | - |
| HALF (2분음표) | 60점 이상 | - |
| WHOLE (온음표) | 100점 이상 | - |

**갭 (미구현 / 불완전)**
- 전화번호 인증 UI 없음
- 회원 탈퇴 후 재가입 플로우 미정의
- 차단 목록 UI 페이지 없음
- 이메일 인증 완료 여부 체크 로직 일부 불일치(버그 이력)

---

### 3.2 게시물 (Posts)

**구현 완료**

카테고리 체계:

| 구분 | 카테고리 |
|---|---|
| 구인구직 | JOB_OFFER (반주자 구함), JOB_SEEK (반주자 구해요), LESSON_OFFER (레슨 제공), LESSON_SEEK (레슨 받고싶어요) |
| 공연/행사 | PERFORMANCE, AFTERSCHOOL, ETC |
| 홍보 | PROMO_CONCERT (공연 홍보), PROMO_SPACE (연습실 홍보) |
| 거래 | TRADE_LESSON, TRADE_SPACE, TRADE_TICKET, TRADE_INSTRUMENT (중고악기) |

핵심 필드: 악기, 지역, 보수(HOURLY/PER_SESSION/MONTHLY/NEGOTIABLE), 이미지, 이벤트 날짜

**프리미엄 노출 (결제 연동)**
- 1일(₩10,000), 7일(₩50,000), 30일(₩150,000)
- `isPremium`, `premiumExpiresAt` 필드로 상태 관리
- 만료 자동 처리: `posts.scheduler.ts`

**갭**
- 공고 검색: 필터(악기·지역·카테고리) UI 완성도 확인 필요
- 공고 만료 정책(14일 자동 마감 등) 명문화 필요
- TRADE 카테고리 별도 UX 분리 여부 미결정

---

### 3.3 지원 (Applications)

**구현 완료**
- 공고에 지원 → 채팅방 자동 생성 (ChatModule 의존)
- 지원 수락/거절
- 지원 시 알림 발송 (NotificationsModule 의존)

**갭**
- 지원 상태 변경 알림 (수락/거절) UI 피드백 확인 필요
- 지원자 목록 페이지 완성도 미확인

---

### 3.4 채팅 (Chat)

**구현 완료**
- 1:1 채팅방 (`chat-room.entity.ts`, `chat-message.entity.ts`)
- Socket.io 실시간 메시지 (`chat.gateway.ts`)
- 지원 수락 시 자동 채팅방 개설

**갭**
- 채팅 읽음 표시(read receipt) 구현 여부 미확인
- 채팅 내 이미지/파일 전송 기능 없음
- 채팅방 목록 정렬(최신 메시지 기준) 완성도 확인 필요

---

### 3.5 커뮤니티 게시판 (Board)

**구현 완료**
- 게시글 + 댓글 + 해시태그
- 유저 생성 태그, 인기 태그, 정렬
- 좋아요 (`likeCount` 컬럼)

**갭**
- 태그 기반 피드 필터 UI 완성도 확인 필요
- 댓글 대댓글(nested) 지원 여부 미확인

---

### 3.6 알림 (Notifications)

**구현 완료**
- FCM 웹 푸시 (`fcm.service.ts`)
- 인앱 알림 (`notification.entity.ts`)
- 알림 설정 (`notification-setting.entity.ts`)
- **키워드 알림**: 사용자 등록 키워드 포함 공고 자동 알림 (`keyword.entity.ts`)

**갭**
- 알림 읽음 일괄 처리 UI 확인 필요
- 키워드 알림 발송 트리거 로직(게시물 생성 시) 완성도 확인 필요
- iOS Safari FCM 지원 제한 대응 방안 없음

---

### 3.7 결제 (Payments)

**구현 완료**
- 토스페이먼츠 연동
- 프리미엄 게시물 상승 노출 (1일/7일/30일)
- 결제 상태: PENDING → SUCCESS / FAILED / CANCELLED / REFUNDED
- 환불 사유 저장

**갭**
- 결제 내역 마이페이지 UI 확인 필요
- 환불 처리 관리자 워크플로우 완성도 확인 필요
- 영수증/세금계산서 발행 기능 없음

---

### 3.8 매칭 후기 (Reviews)

**구현 완료** _(최근 추가, 2025-06 커밋)_
- 리뷰어 → 리뷰이, 공고 연결, 별점(1~5), 텍스트, 공개 여부
- 신뢰도 점수(`trustScore`) 반영 → 등급 승급 알림

**갭**
- 리뷰 작성 트리거 (매칭 완료 후 리뷰 요청 알림) 확인 필요
- 리뷰 중복 방지 로직 확인 필요
- 프로필 페이지에서 리뷰 목록 표시 UI 확인 필요

---

### 3.9 출석체크 (Attendance)

**구현 완료** _(최근 추가, 2025-06 커밋)_
- 일별 출석 체크 → 신뢰도 포인트 적립

**갭**
- 연속 출석 보너스 정책 미정의
- 출석 기록 UI(달력 뷰 등) 없음

---

### 3.10 QnA / 신고 / 관리자

**구현 완료**
- QnA 문의 시스템
- 신고(Reports) 접수
- 관리자 페이지: 회원/공고/신고/QnA/결제 관리

**갭**
- 신고 처리 후 신고자 피드백(알림) 없음
- 관리자 통계 대시보드(가입자 수, 매칭 수, 매출 등) 없음
